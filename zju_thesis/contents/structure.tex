\chapter{学位论文基本结构}
学位论文基本结构包括前置部份、主体部份和结尾部份。
\section{前置部分包括}
\begin{enumerate}
	\item 封面
	\item 题名页
	\item 英文题名页（硕士可省略）
	\item 独创性声明（知识产权声明？）
	\item 勘误表（可根据需要）
	\item 致谢
	\item 序言或前沿（可根据需要）
	\item 摘要页
	\item 目次页
	\item 插图和附表清单（可根据需要）
	\item 缩写、符号清单、术语表（可根据需要）
\end{enumerate}
